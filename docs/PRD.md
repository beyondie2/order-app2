# 커피 주문 앱
## 1. 프로젝트 개요
### 1.1 프로젝트명
커피 주문 앱
### 1.2 프로젝트 목적
사용자가 커피 메뉴를 주문하고, 관리자가 주문을 관리할 수 있는 간단한 풀스택 웹 앱
### 1.3 개발 범위
- 주문하기 화면(메뉴 선택 및 장바구니 기능)
- 관리자 화면(재고 관리 및 주문 상태 관리)
- 데이터를 생성/조회/수정/삭제할 수 있는 기능
## 2. 기술 스택
- 프런트엔드: HTML, CSS, 리액트, 자바스크립트
- 백엔드: Node.js, Express
- 데이터베이스: PostgreSQL
## 3. 기본 사항
- 프런트엔드와 백엔드를 따로 개발
- 기본적인 웹 기술만 사용
- 학습 목적이므로 사용자 인증이나 결제 기능은 제외
- 메뉴는 커피 메뉴만 있음

## 4. 화면 정의서

### 4.1 주문하기 화면

#### 4.1.1 화면 개요
- **화면명**: 주문하기
- **화면 설명**: 사용자가 커피 메뉴를 조회하고, 옵션을 선택하여 장바구니에 담은 후 주문을 완료하는 화면
- **진입 경로**: 앱 실행 시 기본 화면 / 헤더의 "주문하기" 버튼 클릭

#### 4.1.2 화면 구성

##### A. 헤더 영역
| 요소 | 설명 |
|------|------|
| 로고 | 좌측에 "COZY" 브랜드명 표시 |
| 주문하기 버튼 | 주문하기 화면으로 이동 (현재 화면일 경우 활성화 상태) |
| 관리자 버튼 | 관리자 화면으로 이동 |

##### B. 메뉴 목록 영역
| 요소 | 설명 |
|------|------|
| 레이아웃 | 메뉴 카드를 그리드 형태로 가로 배열 |
| 메뉴 카드 | 각 메뉴별로 카드 형태로 표시 |

##### C. 메뉴 카드 상세
| 요소 | 설명 |
|------|------|
| 상품 이미지 | 메뉴 이미지 표시 영역 |
| 상품명 | 메뉴 이름 (예: 아메리카노(ICE), 아메리카노(HOT), 카페라떼) |
| 가격 | 메뉴 기본 가격 (예: 4,000원, 5,000원) |
| 설명 | 메뉴에 대한 간단한 설명 |
| 옵션 선택 | 체크박스 형태의 추가 옵션 선택 영역 |
| 담기 버튼 | 선택한 메뉴와 옵션을 장바구니에 추가 |

##### D. 옵션 항목
| 옵션명 | 추가 금액 | 설명 |
|--------|-----------|------|
| 샷 추가 | +500원 | 에스프레소 샷 1개 추가 |
| 시럽 추가 | +0원 | 시럽 추가 (무료) |

##### E. 장바구니 영역
| 요소 | 설명 |
|------|------|
| 제목 | "장바구니" 텍스트 표시 |
| 장바구니 아이템 목록 | 담긴 메뉴명, 선택 옵션, 수량, 금액 표시 |
| 총 금액 | 장바구니 내 모든 아이템의 합계 금액 |
| 주문하기 버튼 | 주문 완료 처리 |

#### 4.1.3 기능 요구사항

##### 메뉴 조회
- [ ] 등록된 모든 메뉴를 카드 형태로 표시한다
- [ ] 각 메뉴의 이미지, 이름, 가격, 설명을 표시한다
- [ ] 각 메뉴에서 선택 가능한 옵션을 체크박스로 표시한다

##### 장바구니 담기
- [ ] "담기" 버튼 클릭 시 선택한 메뉴와 옵션이 장바구니에 추가된다
- [ ] 동일한 메뉴+옵션 조합을 다시 담으면 수량이 증가한다
- [ ] 옵션 선택에 따라 가격이 자동 계산된다

##### 장바구니 관리
- [ ] 장바구니에 담긴 아이템 목록을 표시한다
- [ ] 각 아이템의 메뉴명, 선택 옵션, 수량, 금액을 표시한다
- [ ] 총 금액을 실시간으로 계산하여 표시한다

##### 주문하기
- [ ] "주문하기" 버튼 클릭 시 주문이 생성된다
- [ ] 주문 완료 후 장바구니가 초기화된다

#### 4.1.4 화면 예시 데이터

**메뉴 목록 예시**
| 메뉴명 | 가격 |
|--------|------|
| 아메리카노(ICE) | 4,000원 |
| 아메리카노(HOT) | 4,000원 |
| 카페라떼 | 5,000원 |

**장바구니 예시**
| 메뉴명 | 옵션 | 수량 | 금액 |
|--------|------|------|------|
| 아메리카노(ICE) | 샷 추가 | 1 | 4,500원 |
| 아메리카노(HOT) | - | 2 | 8,000원 |
| **총 금액** | | | **12,500원** |

### 4.2 관리자 화면

#### 4.2.1 화면 개요
- **화면명**: 관리자
- **화면 설명**: 관리자가 주문 현황을 확인하고, 주문 상태를 변경하며, 메뉴별 재고를 관리하는 화면
- **진입 경로**: 헤더의 "관리자" 버튼 클릭

#### 4.2.2 화면 구성

##### A. 헤더 영역
| 요소 | 설명 |
|------|------|
| 로고 | 좌측에 "COZY" 브랜드명 표시 |
| 주문하기 버튼 | 주문하기 화면으로 이동 |
| 관리자 버튼 | 관리자 화면으로 이동 (현재 화면일 경우 활성화 상태) |

##### B. 관리자 대시보드 영역
| 요소 | 설명 |
|------|------|
| 제목 | "관리자 대시보드" 텍스트 표시 |
| 총 주문 | 전체 주문 건수 표시 |
| 주문 접수 | "주문 접수" 상태인 주문 건수 |
| 제조 중 | "제조 중" 상태인 주문 건수 |
| 제조 완료 | "제조 완료" 상태인 주문 건수 |

##### C. 재고 현황 영역
| 요소 | 설명 |
|------|------|
| 제목 | "재고 현황" 텍스트 표시 |
| 재고 카드 | 각 메뉴별로 카드 형태로 표시 |

##### D. 재고 카드 상세
| 요소 | 설명 |
|------|------|
| 메뉴명 | 메뉴 이름 (예: 아메리카노 (ICE), 아메리카노 (HOT), 카페라떼) |
| 재고 수량 | 현재 재고 수량 표시 (예: 10개) |
| + 버튼 | 재고 수량 1개 증가 |
| - 버튼 | 재고 수량 1개 감소 |

##### E. 주문 현황 영역
| 요소 | 설명 |
|------|------|
| 제목 | "주문 현황" 텍스트 표시 |
| 주문 목록 | 접수된 주문 리스트 표시 |

##### F. 주문 아이템 상세
| 요소 | 설명 |
|------|------|
| 주문 일시 | 주문이 접수된 날짜와 시간 (예: 7월 31일 13:00) |
| 메뉴 정보 | 주문한 메뉴명과 수량 (예: 아메리카노(ICE) x 1) |
| 금액 | 해당 주문의 총 금액 (예: 4,000원) |
| 상태 버튼 | 주문 상태를 변경하는 버튼 |

#### 4.2.3 주문 상태 흐름
| 현재 상태 | 버튼 텍스트 | 클릭 시 변경되는 상태 |
|-----------|-------------|----------------------|
| 주문 접수 | "주문 접수" | 제조 중 |
| 제조 중 | "제조 중" | 제조 완료 |
| 제조 완료 | "제조 완료" | (최종 상태) |

#### 4.2.4 기능 요구사항

##### 대시보드 조회
- [ ] 전체 주문 건수를 표시한다
- [ ] 각 상태별(주문 접수, 제조 중, 제조 완료) 주문 건수를 표시한다
- [ ] 주문 상태 변경 시 대시보드 수치가 실시간 업데이트된다

##### 재고 관리
- [ ] 모든 메뉴의 현재 재고 수량을 표시한다
- [ ] + 버튼 클릭 시 해당 메뉴의 재고가 1개 증가한다
- [ ] - 버튼 클릭 시 해당 메뉴의 재고가 1개 감소한다
- [ ] 재고가 0개일 경우 - 버튼은 비활성화된다

##### 주문 현황 조회
- [ ] 모든 주문을 목록 형태로 표시한다
- [ ] 각 주문의 일시, 메뉴 정보, 금액, 현재 상태를 표시한다

##### 주문 상태 변경
- [ ] 상태 버튼 클릭 시 주문 상태가 다음 단계로 변경된다
- [ ] 주문 접수 → 제조 중 → 제조 완료 순서로 진행된다
- [ ] 제조 완료 상태에서는 더 이상 상태 변경이 불가하다

#### 4.2.5 화면 예시 데이터

**대시보드 예시**
| 항목 | 수치 |
|------|------|
| 총 주문 | 1 |
| 주문 접수 | 1 |
| 제조 중 | 0 |
| 제조 완료 | 0 |

**재고 현황 예시**
| 메뉴명 | 재고 |
|--------|------|
| 아메리카노 (ICE) | 10개 |
| 아메리카노 (HOT) | 10개 |
| 카페라떼 | 10개 |

**주문 현황 예시**
| 주문 일시 | 메뉴 | 금액 | 상태 |
|-----------|------|------|------|
| 7월 31일 13:00 | 아메리카노(ICE) x 1 | 4,000원 | 주문 접수 |
